# Mini E-Commerce

A compact React application demonstrating product listing, filters/search, and a cart with quantity updates. Built with Vite and React.

Features
- Fetches products from https://dummyjson.com/products
- Limits UI to 20 products per page, with pagination controls (search still queries all fetched products)
- Search, category filter, sort by price
- Cart with add/remove/update quantity
- Cart persisted to localStorage
- Debounced search input

Run
1. npm install
2. npm run dev

Notes
- This is a small demo meant to showcase React fundamentals and clean component design.
